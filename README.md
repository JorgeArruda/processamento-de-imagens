# Processamento de Imagens
Implementação de alguns algoritmos para processamento de imagens:
* Operações aritméticas
* Operações lógicas
* Equalização (histograma)
* Passa-baixa
* Passa-alta
* Mediana
* Gaussiana
* Sobel
* Prewitt
* Roberts
* Isotrópico
* Dithering
* Segmentação por limiarização iterativa
